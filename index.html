Projeto 1 - Cadastro Automatizado

Cadastro Automatizado

Este projeto automatiza o preenchimento de um site fictício com dados extraídos de uma planilha Excel. O programa é capaz de ler as informações diretamente da planilha e preenchê-las automaticamente nos campos correspondentes do site, tornando o processo rápido e eficiente.
Projeto 2 - BOT Instagram

BOT Curtidas Instagram

Este programa automatiza o processo de interação com perfis nas redes sociais. O usuário fornece o perfil desejado, e o script verifica se a última publicação foi curtida. Caso a publicação ainda não tenha sido curtida, o programa realiza a ação automaticamente. Se já estiver curtida, o script simplesmente fecha a publicação e finaliza o processo, sem a necessidade de qualquer intervenção manual.

Gerador de Senhas (GUI)
